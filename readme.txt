== Installation ==

Minimum requirements:    
    
 Wordpress 3.0+    
 PHP 5.x    
 MySQL 4.x+  

Installation from within backend:

1. In plugin manager, click "Add New".
2. Search for "os-ansPress-custom-fields".
3. Click "install".
4. Activate the plugin once it is installed.
5. Click on "OS AnsPress Custom Fields" from the left menu and follow the instructions to complete the installation.

FTP upload installation method:

1. Upload the `os-ansPress-custom-fields` folder to the `/wp-content/plugins/` directory using your FTP client.
2. Activate the plugin through the 'Plugins' menu.
3. Go to the OS AnsPress Custom Fields menu, finish the installation.

Zip upload installation method (make sure uploads up to 5MB are allowed for your hosting):

1. Login to your WordPress site administrator panel and head over the 'Plugins' menu.  
2. Click 'Add New'.
3. Choose the 'Upload' option.
4. Click **Choose file** (**Browse**) and select the os-ansPress-custom-fields.*.zip file.   
5. Click **Install Now** button.    
6. Once it is complete, activate the plugin.   
7. Go to the OS AnsPress Custom Fields menu, finish the installation.
